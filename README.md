# 📦 ValidAPP

O **ValidAPP** é um aplicativo desenvolvido para ajudar no controle de validade de produtos de forma simples, rápida e eficiente.

Com ele, você pode cadastrar produtos, acompanhar datas de vencimento e evitar desperdícios no dia a dia.

---

## 🚀 Funcionalidades

* 📅 Cadastro de produtos com data de validade
* 📷 Captura de fotos dos produtos
* 🔍 Leitura de código de barras
* 🖼️ Busca automática de imagens dos produtos
* ☁️ Sincronização de dados com Firebase
* 🔐 Sistema de login para proteção das informações

---

## 📱 Objetivo

O ValidAPP foi criado para facilitar a organização de produtos, ajudando usuários a manterem controle sobre prazos de validade, seja em casa ou no trabalho.

---

## 🛠️ Tecnologias utilizadas

* Python
* Kivy / KivyMD
* Firebase (Authentication e Realtime Database)
